Internally, this is a slightly modified CanvasContext2D object, hence for further information on how the functions work, the canvas documentation can be consulted

fillStyle

fillStyle: string | any
  • Defined in emulator/emulator.api.ts:95

The colour/style of any filling.

The colour can be in HTML form, e.g. "#FF0000" for red, or in rgba form, e.g. "rgba(0.1, 0.2, 0.3, 0.6)". Other css styles are also supported.

font

font: string
  • Defined in emulator/emulator.api.ts:110

The type face for any text.

E.g. "15px sans-serif" for a font of 15px size, in a sans-serif font.

strokeStyle

strokeStyle: string | any
  • Defined in emulator/emulator.api.ts:103

The colour/style of any stroking/lines.

The colour can be in HTML form, e.g. "#FF0000" for red, or in rgba form, e.g. "rgba(0.1, 0.2, 0.3, 0.6)". Other css styles are also supported.

textAlign

textAlign: string
  • Defined in emulator/emulator.api.ts:118

Allignment of text relative to defined position.

Options are "right", "left", "center", which put the specified text coordinate at the given position relative to the text.
