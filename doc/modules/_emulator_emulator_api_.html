External module "emulator/emulator.api"

Emulator Interface.

This contains several interfaces

  • The emulator interface. The public methods on this define the public interface for an application running inside the emulator.
  • The graphics context for drawing on the watch screen.
  • The emulator_storage class, which is how the application can access persistant storage on the smart watch.
